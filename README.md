## Getting Started

### Pré-requisitos

É necessário ter Npm, Node e Mysql instalados em seu computador

### Installation

1. Clonar o repositório
   ```sh
   git clone https://guiqsassi/Real-Fast-Health.git
   ```
2. Instalar os pacotes
   ```sh
   npm i 
   ```
   ```sh
   cd /App 
   ```
   ```sh
   npm i 
   ```
